# Privacy Policy

**Last updated:** _17-12-2025_

## 1. Data Collection
The Bot collects **only the minimum data necessary** to function via slash commands.

This may include:
- Discord user IDs
- Discord server (guild) IDs
- Channel IDs
- Slash command interaction data
- Server or user configuration settings

The Bot **does not read, log, or store regular message content**.

---

## 2. Slash Commands Only
The Bot operates **exclusively using Discord slash commands**.  
It does not monitor messages, DMs, or conversations outside of these interactions.

---

## 3. Use of Collected Data
Collected data is used solely to:
- Execute slash commands
- Store preferences or configuration
- Improve Bot stability and features

Data is **never used for advertising, profiling, or tracking**.

---

## 4. Third-Party APIs
Some slash commands may send limited data to third-party APIs to function.

These services operate under their own privacy policies and licenses (including the Apache License 2.0 where applicable). We do not control how third-party services handle data.

---

## 5. Open-Source Transparency
Because the Bot is open source, data handling practices may be reviewed in the public source code repository.  
Self-hosted instances may differ in behavior and data handling.

---

## 6. Data Retention
- Data is stored only as long as required for functionality
- Unused or obsolete data may be periodically deleted
- No sensitive personal information is intentionally stored

---

## 7. Data Sharing
We **do not sell, trade, or rent** user data.

Data may be shared only:
- When required for third-party API functionality
- When legally required

---

- Information about stored data
- Deletion of stored data

## 9. Security
Reasonable security measures are used to protect stored data. However, no system can guarantee absolute security.

---

## 10. Children’s Privacy
The Bot is not intended for users under the age of 13, in compliance with Discord’s Terms of Service.

---

This Privacy Policy may be updated at any time. Continued use of the Bot indicates acceptance of the updated policy.
